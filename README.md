# Industry Momentum Strategies

This repository implements industry momentum strategies using daily
Fama–French 48 Industry Portfolio data.

## Structure
- `data/`: Raw Fama–French daily industry returns
- `src/`: Strategy and benchmark construction
- `notebooks/`: Figures and tables used in the paper

## Strategies
- Benchmark: Monthly equal-weighted industry portfolio
- 12–0 Momentum
- 12–1 Momentum

Sample period: 1975–2024
